# 교육청 어댑터 실험 기록

안정판 redist 생성 뒤, 현재 작업 폴더에만 교육청 어댑터를 실험적으로 추가했다.

## 안정판

실험 코드가 들어가기 전 안정판 ZIP:

```text
C:\Users\Dell\Documents\0_KICE Test Download and Assemble redist_stable_20260518-195752.zip
```

이 ZIP은 `생활과 윤리`, `윤리와 사상` 평가원/EBSi 흐름이 확인된 상태의 스냅샷이다.

## 실험 방식

교육청 학평도 EBSi 고3 기출 AJAX에서 일부 확인 가능했다.

- 2026년 3월 학평(서울) 생활과 윤리: 가능
- 2026년 3월 학평(서울) 윤리와 사상: 가능
- 2026년 4월/7월/10월: 현 시점 미발견

평가원과 다른 점:

- 평가원: 표시 학년도는 시행연도 + 1
- 교육청: 표시 연도는 시행연도 그대로
- 교육청 파일명과 폴더명은 `2026년-3월-학력평가-...` / `2026년/3월/...` 형식을 사용

## 구현

`official_exam_sources.py`에 `--exam-family office` 옵션을 추가했다.

예시:

```powershell
python official_exam_sources.py --academic-years 2026 --sessions 3월 4월 7월 10월 --subject-id 63002 --subject-name "생활과 윤리" --area-ord 5 --target-cd D300 --exam-family office --out reports/office_adapter_probe_생활과_윤리_2026.csv
```

`subjects.json`에는 실험 과목을 추가했다.

- 교육청 생활과 윤리
- 교육청 윤리와 사상
- 교육청 통합사회: 공식 루트 확인 전까지 비활성

2026-05-19 패치:

- UI에서 `시험 구분`을 먼저 고르고, 그 안에서 사회탐구 과목을 고르는 방식으로 변경했다.
- 교육청 갈래에는 평가원 갈래와 맞춰 사회탐구 과목을 가능한 만큼 노출한다.
- 활성 과목: 생활과 윤리, 윤리와 사상, 한국지리, 세계지리, 동아시아사, 세계사, 정치와 법, 경제, 사회·문화.
- 교육청 통합사회는 앞으로 공식 루트가 열리면 `subject_id`, `area_ord`, `target_cd`를 채워 활성화하는 준비 항목으로 둔다.
- 내부 저장 폴더명은 평가원/교육청 파일이 섞이지 않도록 `교육청 생활과 윤리`처럼 고유 이름을 유지한다.
- 과목별 원본/복사 경로가 아직 지정되지 않은 경우에는 도구 폴더 내부의 `downloads/<과목>/current`, `exports/<과목>`를 기본 경로로 사용한다.
