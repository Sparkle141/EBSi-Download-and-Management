# 버튼 기능 확인 기록

확인일: 2026-05-18

## UI 동작

- 가능항목 표는 다중 선택을 지원한다.
- `Ctrl` 또는 `Shift`를 사용해 여러 행을 선택할 수 있다.
- `가능 항목 전체 선택` 버튼으로 현재 다운로드 가능한 행을 한 번에 선택할 수 있다.
- `선택 해제` 버튼으로 선택을 지울 수 있다.
- 표 머리글을 클릭하면 `학년도`, `회차`, `문서`, `상태`, `공식 제목` 기준으로 재정렬된다.
- `버튼 기능 안내` 버튼으로 각 버튼의 역할을 확인할 수 있다.
- 선택 다운로드 이후 절차는 가능여부 목록 아래 `선택 이후 절차` 영역에 배치했다.

## 다운로드 범위 기준

기본 범위는 `archive_config.json`의 `academic_year_start`와 `academic_year_end`를 따른다.

`다음 학년도까지 확인`을 켜면 현재 날짜 기준으로 평가원/EBSi 범위가 `현재 연도 + 1 학년도`까지 늘어난다. 예를 들어 2026년에 실행하면 평가원 시험은 2027학년도까지 확인한다.

교육청 시험은 평가원과 연도 명칭 체계가 다르므로 별도 어댑터에서 공식 제목과 시행연도를 함께 보존하는 방식으로 확장한다.
