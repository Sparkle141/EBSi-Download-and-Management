# Claude Guide

Claude Code 또는 Claude Cowork에서 이 폴더를 열었다면 루트의 `AGENTS.md`를 먼저 읽고 따르세요.

이 프로젝트에서 특히 중요한 원칙은 다음입니다.

- 실제 시험 문제, 빠른 정답 이미지, 해설 PDF, 영어 듣기 MP3, 대본 PDF를 커밋하지 않습니다.
- 다운로드와 저장 경로 복사는 사용자 로컬 환경에서만 수행합니다.
- 영어 듣기 음성은 전사하지 않고, 제공 대본 PDF를 사용합니다.
- AI Agent 검수 시 문제 PDF, 해설 PDF, 영어 대본 PDF를 문항 단위로 맞추되, 결과물에는 한계와 처리 근거를 `AI metadata`로 남깁니다.
- 코드 변경 후에는 `AGENTS.md`의 검증 명령을 우선 실행합니다.

사용자-facing 문서는 한국어를 기본으로 작성하세요.
